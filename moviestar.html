MOVIESTAR


Este é um projeto particular de estudo, se trata de uma rede social de filmes
onde é possivel cadastrar um usuário com upload de foto, filmes e fazer todas as operações de CRUD
e também é possivel dar notas aos filmes e fazer resenhas e comentarios em outros filmes de outros usuários
cadastrados no sistema. Eu atuei na construção do layout, na estruturação de todo o projeto e fiquei a cargo de toda a produção,
desenvolvendo todo o código do zero sem o uso de frameworks.

As técnologias utilizadas neste projeto foram:

  • HTML
  • CSS
  • Bootstrap
  • PHP
  • MySql
